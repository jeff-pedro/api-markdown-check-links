# API Markdown Check Links
This API validate URLs in markdown files. 

## Install
```
npm install
```

## Use

### Validate
```
./cli.js --path PATH --validate 
```
or
```
npm run cli:validate PATH
```

### List links
```
./cli.js --path PATH
```
or
```
npm run cli PATH
```

### For Help!
```
./cli.js --help
```
or
```
npm run cli:help
```

## Technologies used
- **yargs**:  used to parse cli arguments to objects
- **chalk**:  to color and give more life :smiley:
